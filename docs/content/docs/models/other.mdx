WinSTT supports several additional model families through the `onnx_asr` backend. These are primarily focused on Russian language support.

## GigaAM Models

GigaAM is a family of Russian-language ASR models. All GigaAM models support realtime transcription and use the `onnx_asr` backend.

| Model | Architecture | Notes |
|-------|-------------|-------|
| `gigaam-v2-ctc` | CTC | Version 2, CTC decoding |
| `gigaam-v2-rnnt` | RNN-T | Version 2, transducer decoding |
| `gigaam-v3-ctc` | CTC | Version 3, CTC decoding |
| `gigaam-v3-rnnt` | RNN-T | Version 3, transducer decoding |
| `gigaam-v3-e2e-ctc` | End-to-End CTC | Version 3, end-to-end CTC |
| `gigaam-v3-e2e-rnnt` | End-to-End RNN-T | Version 3, end-to-end transducer |

## Kaldi/Vosk Models

Vosk provides offline speech recognition models based on the Kaldi toolkit. These are lightweight and run well on CPU.

| Model | Description |
|-------|-------------|
| `alphacep/vosk-model-ru` | Full Russian model. Higher accuracy, larger download |
| `alphacep/vosk-model-small-ru` | Lightweight Russian model. Faster loading, smaller footprint |

## T-One Models

T-One is a Russian ASR model from T-Tech.

| Model | Description |
|-------|-------------|
| `t-tech/t-one` | Russian speech recognition |

<Callout title="Language support" type="info">
  All models on this page are Russian-language only. For English or multilingual transcription, use Whisper or NeMo models.
</Callout>
