WinSTT supports several additional model families on the ONNX engine. These are primarily focused on Russian language support. All support realtime transcription.

## GigaAM Models

GigaAM is a family of Russian-language ASR models.

| Model | Architecture | Notes |
|-------|-------------|-------|
| `gigaam-v3-e2e-ctc` | End-to-End CTC | Version 3, end-to-end CTC (faster decoding) |
| `gigaam-v3-e2e-rnnt` | End-to-End RNN-T | Version 3, end-to-end transducer (higher accuracy) |

GigaAM models support `""` (default) and `int8` via `onnxQuantization`.

## Kaldi/Vosk Models

Vosk provides offline speech recognition models based on the Kaldi toolkit. These are lightweight and run well on CPU.

| Model | Description |
|-------|-------------|
| `alphacep/vosk-model-ru` | Full Russian model. Higher accuracy, larger download |
| `alphacep/vosk-model-small-ru` | Lightweight Russian model. Faster loading, smaller footprint |

Vosk/Kaldi models support `""` (default) and `int8` via `onnxQuantization`.

## T-One Models

T-One is a Russian ASR model from T-Tech.

| Model | Description |
|-------|-------------|
| `t-tech/t-one` | Russian speech recognition. Default precision only (`""`) |

<Callout title="Language support" type="info">
  All models on this page are Russian-language only. For English or multilingual transcription, use Whisper or NeMo models.
</Callout>
