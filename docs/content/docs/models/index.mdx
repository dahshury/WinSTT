WinSTT supports a wide range of speech recognition models across two inference backends.

## Backends

**faster_whisper** -- Runs OpenAI Whisper models using CTranslate2. Supports all Whisper model sizes with configurable quantization. This is the default backend.

**onnx_asr** -- Runs NVIDIA NeMo, GigaAM, Kaldi/Vosk, and T-One models via ONNX Runtime. Provides access to non-Whisper architectures including CTC, RNN-T, and TDT decoders.

Set the backend in [Model Settings](/docs/settings/model) using the `backend` field.
