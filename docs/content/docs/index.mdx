WinSTT is a Windows desktop application for real-time speech-to-text. It pairs a Python backend (WebSocket STT server) with an Electron frontend to deliver fast, accurate transcription directly on your machine.

**All processing runs locally.** No cloud APIs, no internet connection required, no telemetry. Your audio is processed in-memory by on-device AI models and never leaves your computer.

## Privacy by Design

WinSTT is built around a simple principle: **your voice data belongs to you**.

- **Offline-first** — transcription works without an internet connection. Audio never leaves your machine.
- **In-memory processing** — audio is transcribed in real-time and discarded. Nothing is written to disk unless you explicitly export or enable transcription history.
- **No analytics tracking** — zero usage analytics. The only outbound signal is anonymized **crash reports** (Sentry), which are opt-out and can be disabled in [General Settings](/docs/settings/general).
- **Optional cloud LLM** — text enhancement runs locally via Ollama by default; OpenRouter is available only if you explicitly opt in and supply an API key.
- **Open source** — every line of code is auditable. You can verify exactly what runs on your machine.
