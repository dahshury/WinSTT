WinSTT is built on a lot of generous open-source work. The full per-file license text is in [THIRD_PARTY_NOTICES.md](https://github.com/dahshury/WinSTT/blob/main/THIRD_PARTY_NOTICES.md); this page is the friendly summary of who to thank.

## Speech recognition models

- **[OpenAI Whisper](https://github.com/openai/whisper)** — base, small, medium, large-v3-turbo, and the English-only `.en` variants. MIT.
- **[Lite-Whisper](https://huggingface.co/efficient-speech)** — compressed Whisper variants (`-acc`, `-fast`). Apache 2.0 / MIT.
- **[NVIDIA NeMo](https://github.com/NVIDIA/NeMo)** — Parakeet TDT (English + multilingual) and Canary. Apache 2.0; CC BY 4.0 on model weights.
- **[Moonshine](https://github.com/usefulsensors/moonshine)** (Useful Sensors) — efficient ASR for edge devices. MIT.
- **[Cohere](https://huggingface.co/CohereForAI)** — open-weight ASR for low-resource languages. CC BY-NC 4.0 on model weights — please check the license for your use case.
- **[GigaAM](https://github.com/salute-developers/GigaAM)** — Russian ASR. Apache 2.0.
- **[Vosk / Kaldi](https://github.com/alphacep/vosk-api)** — Russian small-footprint models. Apache 2.0.
- **[T-One](https://github.com/t-tech/t-one)** — Russian ASR. Apache 2.0.
- **[onnx-community / Xenova](https://huggingface.co/onnx-community)** — the heroes maintaining ONNX exports of every model above.

## Voice activity & wake word

- **[Silero VAD](https://github.com/snakers4/silero-vad)** — neural voice-activity detection. MIT.
- **[WebRTC VAD](https://webrtc.org/)** — fast classical VAD for the pre-check stage. BSD.
- **[Picovoice Porcupine](https://picovoice.ai/platform/porcupine/)** — free-tier wake-word engine. Apache 2.0 SDK; built-in keywords under Picovoice's free-tier license.

## Text-to-speech

- **[Kokoro-82M](https://huggingface.co/hexgrad/Kokoro-82M)** — the bundled ONNX TTS voice model. Apache 2.0.
- **[kokoro-onnx](https://github.com/thewh1teagle/kokoro-onnx)** — Python wrapper, torch-free. MIT.

## Inference & audio runtime

- **[ONNX Runtime](https://onnxruntime.ai/)** — the single inference engine WinSTT ships. MIT.
- **[DirectML](https://github.com/microsoft/DirectML)** — the EP that makes the default GPU build vendor-agnostic. MIT.
- **[PortAudio](http://www.portaudio.com/)** — cross-platform audio I/O. MIT-style.
- **[onnx-asr](https://github.com/istupakov/onnx-asr)** — the ASR abstraction layer WinSTT depends on (we ship a [WinSTT-side fork](https://github.com/dahshury/onnx-asr) pinned by commit). MIT.
