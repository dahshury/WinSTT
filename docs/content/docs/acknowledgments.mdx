WinSTT stands on a lot of generous open-source work. This page is the friendly summary of who to thank; the full per-file license text lives in [THIRD_PARTY_NOTICES.md](https://github.com/dahshury/WinSTT/blob/main/THIRD_PARTY_NOTICES.md).

<Callout type="info" title="Check the model licenses for your use case">
  WinSTT is MIT, but the **models** it can download carry their own terms. Most are Apache 2.0 or MIT, but a few (Cohere weights, NeMo weights) are CC BY-NC / CC BY — fine for personal use, but read the upstream license before shipping anything commercial.
</Callout>

## Voice activity & wake word

- **[Silero VAD](https://github.com/snakers4/silero-vad)** — neural voice-activity detection. MIT.
- **[WebRTC VAD](https://webrtc.org/)** — fast classical VAD for the pre-check stage. BSD.
- **[Picovoice Porcupine](https://picovoice.ai/platform/porcupine/)** — free-tier wake-word engine. Apache 2.0 SDK; built-in keywords under Picovoice's free-tier license.

## Text-to-speech

- **[Kokoro-82M](https://huggingface.co/hexgrad/Kokoro-82M)** — the bundled ONNX TTS voice model. Apache 2.0.
- **[kokoro-onnx](https://github.com/thewh1teagle/kokoro-onnx)** — reference implementation that helped validate the ONNX voice model. MIT.

## Inference & audio runtime

- **[ONNX Runtime](https://onnxruntime.ai/)** — the single inference engine WinSTT ships. MIT.
- **[DirectML](https://github.com/microsoft/DirectML)** — the execution provider that makes the default GPU build vendor-agnostic (AMD / Intel / NVIDIA via DirectX 12). MIT.
- **[OpenVINO](https://github.com/openvinotoolkit/openvino)** — the EP for the Intel-tuned build (ARC dGPU / Iris Xe). Apache 2.0.
- **[PortAudio](http://www.portaudio.com/)** — cross-platform audio I/O. MIT-style.
- **[onnx-asr](https://github.com/istupakov/onnx-asr)** — reference ASR implementation used by the original Electron app and retained for parity benchmarking. MIT.

## Frontend & desktop

- **[Tauri](https://tauri.app/)** — desktop runtime and WebView shell. MIT / Apache 2.0.
- **[Vite](https://vitejs.dev/)** — renderer build tool. MIT.
- **[React](https://react.dev/)** + **[React Compiler](https://react.dev/learn/react-compiler)** — UI library + Babel plugin that frees the renderer from `useMemo` / `useCallback`. MIT.
- **[Base UI](https://base-ui.com/)** — accessible unstyled component primitives. MIT.
- **[Hugeicons](https://hugeicons.com/)** — icon set. Free icons under their free license.
- **[Zustand](https://github.com/pmndrs/zustand)** — state stores. MIT.
- **[use-intl](https://www.npmjs.com/package/use-intl)** — i18n runtime. MIT.
- **[uiohook-napi](https://github.com/SnosMe/uiohook-napi)** — global keyboard hook for Push-to-Talk / Toggle. MIT.

## Backend & build tooling

- **[Rust](https://www.rust-lang.org/)** — native backend language and toolchain. MIT / Apache 2.0.
- **[Cargo](https://doc.rust-lang.org/cargo/)** — Rust package manager and build runner. MIT / Apache 2.0.
- **[hf-hub](https://github.com/huggingface/hf-hub)** — Hugging Face model download / cache layer. MIT / Apache 2.0.
- **[Bun](https://bun.sh/)** — JavaScript runtime + package manager. MIT.
- **[ESLint](https://eslint.org/)** + **[typescript-eslint](https://typescript-eslint.io/)** — TypeScript linting. MIT.
- **[Sentry](https://sentry.io/)** — opt-out crash reporting. BSL.

## Documentation

- **[Fumadocs](https://fumadocs.vercel.app/)** — the framework powering this site. MIT.

## Inspiration & adjacent projects

- **[Handy](https://github.com/cjpais/Handy)** by CJ Pais — a Tauri-based local STT app whose docs structure and several settings (mic-lifecycle, paste pipeline) shaped these pages. MIT.
- **[RealtimeSTT](https://github.com/KoljaB/RealtimeSTT)** — one of the upstream projects that informed the original WinSTT reference app. MIT.
- **[Whisper.cpp](https://github.com/ggerganov/whisper.cpp)** — the project that proved Whisper could run efficiently on CPU. MIT.
