---
title: Settings Overview
description: Configure WinSTT to match your needs
---

WinSTT settings are organized into eight panels. Each panel groups related options so you can quickly find what you need.

<Cards>
  <Card title="Model Settings" href="/docs/settings/model">
    Choose transcription models, set precision via ONNX quantization, and tune beam search parameters.
  </Card>
  <Card title="Audio Settings" href="/docs/settings/audio">
    Select your microphone, configure sample rate, and adjust voice activity detection sensitivity.
  </Card>
  <Card title="Quality & Transcription" href="/docs/settings/quality">
    Control realtime preview behavior, batch sizes, endpoint detection, and text formatting rules.
  </Card>
  <Card title="General Settings" href="/docs/settings/general">
    Set application behavior, recording mode, wake word, audio ducking, overlay, and file transcription.
  </Card>
  <Card title="Hotkey Settings" href="/docs/settings/hotkey">
    Configure the global push-to-talk keyboard shortcut.
  </Card>
  <Card title="LLM Enhancement" href="/docs/settings/llm">
    Clean up dictated text and run hotkey-driven text transforms via local or cloud LLMs.
  </Card>
  <Card title="Text-to-Speech" href="/docs/settings/tts">
    Read selected text aloud with the bundled Kokoro-82M ONNX voice.
  </Card>
</Cards>

Dictionary & Snippets are configured separately — see [Dictionary](/docs/dictionary).

Settings are persisted automatically when changed. The Electron main process saves them to disk and syncs them to the Python STT server over the control WebSocket channel.
