title: Hotkey Settings
description: Configure the global push-to-talk hotkey

WinSTT uses a global hotkey that works even when the application window is not focused. The hotkey triggers recording according to the active recording mode.

## Settings Reference

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `pushToTalkKey` | string | `"LCtrl+LMeta"` | The global hotkey, specified as a uiohook-style combo string |
| `repasteHotkey` | string | `"LCtrl+LShift+V"` | Exclusive global shortcut that re-pastes your most recent transcription. Empty string disables it. |

## Re-paste Last Transcription

Pressing the **re-paste** shortcut injects your most recent dictation transcription into whatever app currently has focus — handy when a paste landed in the wrong window, or you want the same text again without re-dictating.

Unlike the push-to-talk hotkey (which only *observes* keys), this shortcut is registered **exclusively**: while WinSTT is running the combo is captured system-wide and **only** triggers the re-paste. It will *not* also fire the focused app's own binding for that combo (e.g. `Ctrl+Shift+V` = paste-without-formatting / terminal paste). That is what makes the behaviour deterministic — and why the combo loses its native meaning in every other app while WinSTT runs.

Because it's exclusive, it's fully rebindable. In **Settings → Hotkey**, record a different combo, or press **Clear** to disable the feature entirely (the setting becomes an empty string and no shortcut is registered).

The text is delivered through the same path as dictation auto-paste (per-character typing — your clipboard is never touched), so it behaves the same in terminals, editors, and browsers.

## Combo Format

The hotkey uses a uiohook-style combo string — **not** an Electron accelerator. Tokens are physical keys joined with `+`.

**Modifiers (left/right specific):** `LCtrl`, `RCtrl`, `LShift`, `RShift`, `LAlt`, `RAlt`, `LMeta`

**Other keys:** letters, digits, `F1`–`F24`

### Examples

| Combo | Description |
|-------|-------------|
| `LCtrl+LMeta` | Left Ctrl and Left Windows key (default) |
| `LCtrl+Space` | Left Ctrl and Spacebar |
| `LAlt+LShift+A` | Left Alt, Left Shift, and A |
| `F13` | Function key 13 (useful for dedicated macro keys) |
| `LCtrl+LShift+F9` | Left Ctrl, Left Shift, and F9 |

<Callout title="Choosing a hotkey" type="info">
  Pick a combination that does not conflict with other applications. Left/right-specific modifiers (e.g. `LMeta` rather than a generic `Meta`) reduce conflicts. Keys like F13–F24 are rarely used by other software and make excellent dedicated hotkeys.
</Callout>
