---
title: Hotkey Settings
description: Configure the global push-to-talk hotkey
---

WinSTT uses a global hotkey that works even when the application window is not focused. The hotkey triggers recording according to the active recording mode.

## Settings Reference

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `pushToTalkKey` | string | `"LCtrl+LMeta"` | The global hotkey, specified as a uiohook-style combo string |

## Combo Format

The hotkey uses a uiohook-style combo string — **not** an Electron accelerator. Tokens are physical keys joined with `+`.

**Modifiers (left/right specific):** `LCtrl`, `RCtrl`, `LShift`, `RShift`, `LAlt`, `RAlt`, `LMeta`

**Other keys:** letters, digits, `F1`–`F24`

### Examples

| Combo | Description |
|-------|-------------|
| `LCtrl+LMeta` | Left Ctrl and Left Windows key (default) |
| `LCtrl+Space` | Left Ctrl and Spacebar |
| `LAlt+LShift+A` | Left Alt, Left Shift, and A |
| `F13` | Function key 13 (useful for dedicated macro keys) |
| `LCtrl+LShift+F9` | Left Ctrl, Left Shift, and F9 |

<Callout title="Choosing a hotkey" type="info">
  Pick a combination that does not conflict with other applications. Left/right-specific modifiers (e.g. `LMeta` rather than a generic `Meta`) reduce conflicts. Keys like F13–F24 are rarely used by other software and make excellent dedicated hotkeys.
</Callout>
